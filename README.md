# Inteligência Artificial na Advocacia

Site educativo sobre Inteligência Artificial aplicada à advocacia, com conteúdo da aula do Dr. Gustavo Rocha.

## Conteúdo

- **9 Tópicos Completos** sobre IA na Advocacia
- **3 Módulos Principais**: Conceitos Fundamentais, Aplicação Prática, Ética e Responsabilidade
- **4 Exercícios Práticos** interativos
- **Informações sobre Dr. Gustavo Rocha** e sua experiência

## Arquivos

- `index.html` - Página principal
- `styles.css` - Estilos e design
- `script.js` - Interatividade

## Design

- Paleta de cores: Vermelho (#DC143C) e Preto (#0a0a0a)
- Design responsivo (mobile, tablet, desktop)
- Animações suaves e efeitos visuais

## Como usar

1. Clone este repositório
2. Abra `index.html` em um navegador
3. Ou faça deploy em um servidor web
